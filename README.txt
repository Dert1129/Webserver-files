    To connect to your database:
    Restart visual studio code 
    Click on the server icon on the left status bar
    Specify the name of the server in the box that appears at the top of vscode
    Specify the name of the database in the box after Specifying the name of the server
    
    Leave the rest of the fields empty
    You can now connect to your database in vscode
